Tennis Note 베타 고객지원

고객지원 안내

Tennis Note는 테니스클럽하우스 회원과 코치를 위한 레슨 운영 앱입니다. 베타 테스트 중 문의는 아래 기준으로 접수합니다.

문의 방법

  • 앱 안의 카카오 문의 버튼
  • 테니스클럽하우스 공식 카카오채널
  • 사무실 또는 담당 코치 문의

문의 가능한 내용

  • 로그인 또는 관리자 승인 문제
  • 회원권 결제, 환불, 취소 확인
  • 수업 시간표, 수업 변경 요청, 보강 확인
  • 운동일지, 사진/동영상 업로드, 코치 코멘트 확인
  • 개인정보 열람, 정정, 삭제 요청

처리 기준

결제, 환불, 회원권, 시간표 변경은 관리자 확인 후 처리됩니다. 베타 테스트 중에는 일부 기능이 점검용으로 운영될 수 있습니다.

개인정보

개인정보 처리 기준은 개인정보 처리방침에서 확인할 수 있습니다.
